About TerraHouse

Distributed smart terrarium control system

TerraHouse it's a platform which serve like a distributed monitoring and control system for terrariums or any other process which require simple repeatable actions like light control, misting control, temperature measurement etc. more.

The power of the system consist of independent control units and one main monitoring and control unit. What is really unique and crucial is that each unit use REST api for interfacing and interaction, that means you can put on top of modules what ever front-end you want like android app, java app, c# app and many more.

Arduino to Raspberry pi with REST api

Arduino modules & Raspberry pi main unit

Main idea was to build platform which can people use to control repeatable process tasks, in my case that was control of terarium, I had couple of terrariums, I was lazy to control light manually etc. But this is not another sketch or library for arduino it is concept how to use arduino like distributed control system

Hardware Adaptability

Every arduino which support ethernet shield is usable, and for main server only require is app which handle REST api

Customize Easily

Easy add additional sensors (temperature, humidity, water) actuators (lights, heaters) and monitor them remotely

Parts available

Parts for making this system is available in most countries
